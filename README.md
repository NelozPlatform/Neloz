# Neloz

## Core Utilities

### Neloz AI Agent

An intelligent AI agent designed to analyze market data, token behavior, and trading signals in real time.

It provides insights, alerts, and strategic recommendations to help investors navigate volatile markets with greater confidence.

---

### AI Trader Chat Assistant

A conversational AI built specifically for traders.

Users can ask questions such as:

- Is this token risky?
- Should I enter this market now?
- What are the red flags in this contract?

The AI provides rational guidance and risk insights to help traders make wiser decisions.

---

### Token Risk Scanner

Before buying a token, Neloz scans for potential dangers such as:

- suspicious contract activity
- liquidity risks
- abnormal trading behavior
- developer wallet concentration

This helps traders avoid traps and identify hidden risks early.

---

### Smart Risk Reminder

Neloz automatically warns users before they buy high-risk tokens.

The system highlights:

- extreme volatility
- suspicious tokenomics
- unhealthy liquidity structures

Giving users time to reconsider impulsive decisions.

---

### Post Purchase Risk Alarm

After a token is purchased, Neloz continuously monitors it.

If unusual activity appears such as:

- sudden liquidity removal
- abnormal wallet movement
- potential rug indicators

The system immediately sends real-time alerts.
